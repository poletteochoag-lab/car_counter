# Smart Lane Car Counter

A lightweight, console-based traffic monitoring system developed in **C++**. This project simulates a real-world vehicle counter for a single lane without using external libraries like OpenCV.

## Features
- **Data Persistence:** Automatically saves and loads the car count from a `.txt` file so you don't lose progress.
- **Object-Oriented Programming:** Built using classes to keep the logic organized and professional.
- **Hardware Ready:** The logic is designed to be easily integrated with physical sensors (Infrared or Ultrasonic) in the future.
